## STEP 1. Overview of the data _Example: Compare the percentage of B cells between grafts and spleen._
You may have some data that looks like this: | grafts | spleen | |:------:|:------:| | 2 | 6 | | 4 | 8 | | 5 | 4 | In this example, we are comparing grafts vs spleen (alternatively this could be control vs treatment). These tissues represent our categories (or groups, or independent variables). We are then measuring the percentage of B cells in each tissue (numerical value). These values represent our data (or observations, or dependent variables).
## STEP 2. Enter data into RStudio http://www.sthda.com/english/articles/24-ggpubr-publication-ready-plots/76-add-p-values-and-significance-levels-to-ggplots/#compare-two-paired-samples http://www.sthda.com/english/wiki/paired-samples-t-test-in-r Let's make group names for each tissue (grafts, spleen), and then feed in the corresponding data for each group: ```{r} grafts <-c(2, 4, 5) spleen <-c(6, 8, 4) ```
Now let's put this information into a table (a tall table, or dataframe). We are going to call the table (dataframe): table.tall ```{r} table.tall <- data.frame( tissue = rep(c("grafts", "spleen"), each = 3), percentB = c(grafts, spleen) ) ``` ```{r} print(table.tall) ``` As you can see, each column **header** in the table represents a _variable_ (eg. category [independent variable] or data [dependent variable]). RStudio likes dataframes in this format.
## STEP 3. Append data into RStudio https://www.datamentor.io/r-programming/data-frame/ To add extra data, append the dataframe. ```{r, eval = TRUE} # Expt FP98 (10/5/23) table.tall <-rbind(table.tall,list("grafts",3)) # Expt FP99 (17/5/23) table.tall <-rbind(table.tall,list("spleen",7)) print(table.tall) ``` _Note: it is useful to add comments to include metadata (ie. which experiment the data originates from)_
